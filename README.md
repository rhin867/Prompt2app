## How to Use (Free & Unlimited)
1. Open `index.html` in your browser.
2. Click the **Settings (Gear)** icon in the bottom left.
3. Get a free API Key from [OpenRouter.ai](https://openrouter.ai/).
4. Paste the key into settings.
5. Ensure the model is set to `deepseek/deepseek-r1:free` (or any other free model ID).
6. Start building!
